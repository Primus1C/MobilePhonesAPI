Проведение документов

Форма предназначена для выполнения операций проведения и перепроведения документов, в также восстановления последовательностей.

Проведение документов

На данной закладке можно проводить и перепроводить документы выбранных видов, которые не помечены на удаление и попадают в заданный интервал времени.

В список Доступные документы попадут только те документы, для которых у текущего пользователя установлено право Интерактивное проведение.

Флажок Прекращать проведение при возникновении ошибки позволяет выполнить проведение всех доступных документов не обращая внимание на ошибки, либо прекратить проведение при первом же документе, при проведении которого обнаружатся ошибки.

При добавлении или исключении документов из списков поддерживается множественное выделение.

Восстановление последовательностей

На данной закладке можно восстановить последовательности, доступные текущему пользователю. В список попадают те последовательности, у которых для текущего пользователя установлено право Использование.

Поддерживается множественное выделение. Восстанавливаться будут все выделенные последовательности, либо та, на которой установлен курсор.

Для восстановления всех последовательностей необходимо выполнить команду Восстановить все.
