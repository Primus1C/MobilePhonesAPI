Список активных пользователей

Список содержит список пользователей, работающих в данный момент времени в системе 1С:Предприятие.

Для удобства просмотра списка можно использовать сортировку по любой колонке.

С помощью команд меню Все действия можно настроить форму и вывести данные списка в табличный и текстовый документ.

С помощью команды Журнал регистрации открывается журнал регистрации. 

С помощью команды Работа пользователя открывается журнал регистрации с установленным фильтром по текущему пользователю.

Полужирным шрифтом выделена строка, соответствующая тому сеансу работы пользователя, из которого открыт список активных пользователей.
